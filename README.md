# sakislab
# 🧪 Sakis Lab | Where Sound Meets Code

Καλώς ήρθατε στο **Sakis Lab**, ένα προσωπικό digital portfolio και platform hub για τη φιλοξενία μουσικών άλμπουμ, λογισμικού, εγγράφων και ψηφιακών assets.

![Sakis Lab Banner](assets/images/favicon.png)

---

## 🌟 Χαρακτηριστικά (Features)

- 🎧 **Music Discography & Albums:** Ενσωματωμένο audio player με υποστήριξη για οργάνωση κομματιών ανά άλμπουμ και προβολή εξωφύλλων (Covers).
- 💻 **Software & Apps Showcase:** Ενότητα παρουσίασης προγραμμάτων (Python, JavaScript κ.ά.) με απευθείας links για κατέβασμα (.zip, .exe).
- 📄 **Document Center:** Φιλοξενία και προβολή σημειώσεων, οδηγών και αρχείων PDF.
- ☁️ **Upload Center:** Διασύνδεση Drag & Drop για προσομοίωση ανεβάσματος αρχείων.
- 🎨 **Modern Glassmorphism UI:** Σχεδιασμός Dark Mode βασισμένος σε Tailwind CSS.

## 📂 Δομή Project (Directory Structure)

```text
sakislab/
├── index.html              # Η κύρια ιστοσελίδα
├── README.md               # Πληροφορίες του project
└── assets/                 # Φάκελος πολυμέσων & αρχείων
    ├── audio/              # Μουσικά άλμπουμ & MP3s
    │   ├── album-1/
    │   └── album-2/
    ├── downloads/          # Προγράμματα (.zip, .exe)
    ├── documents/          # Έγγραφα (.pdf, .docx)
    └── images/             # Εικόνες & Favicon
```

🛠️ Τεχνολογίες που χρησιμοποιήθηκαν
HTML5 & JavaScript (ES6+)
Tailwind CSS (via CDN)
FontAwesome 6 (για εικονίδια)
GitHub Pages (για δωρεάν hosting)
